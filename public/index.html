Grayic Purchase Popup

Live recent-purchase social-proof popup. The widget below is loading from this same deployment. Add it to any site with one tag in the header:

<script src="https://YOUR-VERCEL-APP/popup.js"
        data-position="bottom-left"
        data-interval="8000"
        data-duration="5000"
        defer></script>

Options: data-position (bottom-left | bottom-right | top-left | top-right), data-interval (gap ms), data-duration (visible ms), data-limit (recent count), data-endpoint (override feed URL).
